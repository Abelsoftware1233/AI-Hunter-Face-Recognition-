De combinatie van deze scripts geeft je de bouwstenen van je AI:
Gezichtsvergelijker: Coderen van gezichten.
Database Index: Opslaan van coderingen en URL's.
Zoekautomatisering: Starten van de zoekopdracht.
De volgende (en moeilijkste) stap is het vullen van de database (indexering), wat het ethisch en legaal gevoelige deel is.
